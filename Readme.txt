Программа для записи потребляемых ресурсов, запущенного пользователем процесса, в файл.
Запись происходит через указанный пользователем промежуток времени в секундах.
В качестве данных процесса записываются в файл:
ID процесса,
использование процессора процессом,
исползование физической и виртуальной памяти,
количество дискриптеров открытых процессом.
